setColor(color=None, type='scalars')[source]

Set the color of the mesh data.

If type == “scalars”, the surface of the mesh is drawn using scalar data. In this case, color should be a name of colormap to be used.

If type == “color”, the surface of the mesh is colored by a single color. In this case, color should be a color string, length 3 RGB sequence.

Parameters:

  • color – See above.

  • type – See above.
