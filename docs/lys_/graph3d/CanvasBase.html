CanvasBase module

class lys.widgets.canvas3d.interface.CanvasBase.CanvasBase3D[source]

Bases: object

Base class for canvas.

A 3D canvas is composed of Data.CanvasData3D. All of these classes inherits CanvasPart3D and added by addCanvasPart().

Users can access all public methods of the classes above from canvas instance.
